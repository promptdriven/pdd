# User Story: Classify prose/empty LLM generation output separately from architecture regressions

## Story
As a developer using PDD, I can see a clear, distinct error when the model returns only planning text or nothing instead of code, so that I know immediately the generation didn’t produce a real implementation and can fix the output shape rather than debugging missing code symbols.
