# User Story: pdd-bug: report recoverable step failures as degraded instead of fatal

## Story
As a developer using `pdd bug`, I can see a clear distinction between a workflow that has stopped and one that is continuing after a partial failure, so that I am not misled into thinking my automated fix attempt has died when it is actually still progressing.
