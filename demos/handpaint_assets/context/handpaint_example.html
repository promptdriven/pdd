Running the Hand Paint Demo

The handpaint.html file is a web application that uses your webcam to let you draw in the air with your hand. Because it needs camera access, you must run it from a local web server for it to work correctly in your browser.

Instructions

  1. Open your terminal:

    Navigate to the root directory of this project (pdd).

  2. Start a local web server:

    Run the following command in your terminal. This command uses Python to start a simple web server. If you have a `pdd` conda environment, it's best to use it.

    conda run -n pdd python -m http.server 8080

    If you don't have Conda, and you have Python 3 installed, you can use:
    python3 -m http.server 8080

  3. Open the demo:

    Once the server is running, click the link below or copy-paste it into your browser's address bar. Your browser will ask for permission to use your camera.

    ./demo/src/handpaint.html

How to Use the Hand Paint Demo

  • To Draw: Extend your index finger and move your hand.
  • To Erase: Open your hand to show your full palm.
